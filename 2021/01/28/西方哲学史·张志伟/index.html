西方哲学史·张志伟

1 智慧的痛苦

西方哲学史分期：

  1. 公元前6世纪到公元6世纪，古代哲学

  2. 公元2世纪到16世纪，中古哲学

  3. 17世纪到19世纪，近代哲学

  4. 20世纪，现代西方哲学

1.1 什么是哲学

不做加法做减法→排除什么不是哲学

  • 关于哲学的一些错误看法

    1. 哲学是科学：哲学不具备一门科学知识应该具备的最基本特征：普遍性和必然性。哲学不是自然科学式的科学（自然科学是工具手段，哲学思考工具如何使用、价值取向是什么）

    2. 哲学是使人聪明的学问：哲学思想众说纷纭，糊涂是入门的开始。

    3. 哲学是讲道理的学问：哲学往往被看作是为意识形态服务的工具。

      • 科学知识的对象→现实性→可以经过经验、事实、实验的验证
      • 哲学的对象→理想性→不同的时代、社会、文化背景等等使人们对理想世界的理解不同→而哲学探究的是逻辑，是论证方式，它可以是无懈可击的→具体到个体上大前提可能是有问题的→看似不讲道理

    4. 哲学无用：哲学应该是人的精神家园

      所有学科都比哲学有用，但是唯有哲学是自由的学问。因为所有的学科都有前提，都有它服务的对象，但唯独哲学是为了自身而被追求的知识。——亚里士多德

      人文学科与自然科学学科→工具理性和价值理性的平衡问题→科技社会的异化问题

  • 哲学是什么？philosophia(哲学)=philos+sophia(爱智慧)

    只要那个问题是有可能解决的，它就不再是哲学问题，它就会成为一门学科，而留给哲学的就是无法解决的问题。

    • 科学：研究自然，诉诸于理性。
    • 宗教：关注超自然世界，诉诸于信仰，面对终极关怀（例如人从何而来归于何处，宇宙的本性是什么）的问题。
    • 哲学：和科学一样讲理性，和宗教一样起源于终极关怀的问题。

    哲学问题具有这样的形式：我找不到出路所在。——维特根斯坦

    我们可以把哲学看作是过去、现在乃至将来，围绕哲学问题所做的永恒的探索。哲学=哲学史=问题史

  • 问题与难题：

    • 问题：有标准答案的问题，称之为”问题“。
    • 难题：没有标准答案，只有不同解决方式的问题，称之为”难题“。

    我们面对的大多数问题其实是难题。哲学家面对的是一切难题中的难题。

    为什么非要追问哲学问题？↓

1.2 智慧的痛苦

整个人类发端以来，所有的问题最终归结起来无非是生死的问题。

  • 矛盾：有限/无限、相对/绝对、暂时/永恒

1.3 问题与对话

  • 怎么学哲学？

  1. 了解哲学家的问题。不能知其然而不知其所以然。

  2. 熟悉哲学家的术语。

  3. 沿着哲学家的思路。

  4. 理解哲学家的思想。

1.4 西方哲学精神

  • 希腊人哲学思维的特点：学以致知
  1. 偏爱非个人性的，更关注带有普遍性的东西。
  2. 试图给事物的存在找出合理的原因。

2 哲学的诞生（早期希腊自然哲学）

1.1希腊文明的兴起

希腊哲学发端于公元前6世纪，终止于公元6世纪，前后一千多年。公元529年，皈依了基督教的东罗马帝国皇帝查士丁尼下令封闭了最后一所柏拉图学园，通常人们以这一年作为希腊哲学终结的标志。

希腊哲学历史的跨度：

  • 希腊古典时期
  • 希腊化时期
  • 罗马共和国时期
  • 罗马帝国时期

采用城邦民主制度，最早产生于小亚细亚沿岸的殖民城市，而繁荣是在雅典。

最具代表性的三位希腊哲学家：苏格拉底、柏拉图、亚里士多德。

为什么城邦民主制度是哲学产生的肥沃的土壤？

  • 哲学产生的三个条件：
  1. 人们开始关注终极关怀的问题。
  2. 人们有时间去思考这些问题。
  3. 必须有思想的自由的条件。

城邦林立带来百家争鸣，允许不同的思想出现和碰撞。

1.2 哲学的诞生

​ 从某种意义上说，哲学脱胎于神话。我们讲哲学是世界观，是对世界的整体看法，最早给人提供世界观的不是哲学，而是宗教和神话。

  • 人类最早是以神话的方式给出对世界的说明。例如命运来解释不合理的事情。当人们不满足于用命运进行解释，试图给出一个更合理的说明时，命运便向必然性和规律进行转变。

  • 希腊哲学第一位哲学家：泰勒斯（米利都学派创始人），他的学生阿纳克西曼德：命运的看法向必然性的演变。

    一切存在着的东西都由此生成的也是他们灭亡后的归宿，这是命中注定的。根据时间的安排，它要为各自对他物的损害而互相补偿，得到报应。——阿纳克西曼德

  • 人们不满足于用命运解释自然，试图去研究自然，以自然去解释自然，给自然以合理的解释，哲学由此诞生。

  • 希腊哲学分期：

  1. 前苏格拉底哲学（早期希腊自然哲学、宇宙生成论）
  2. 雅典哲学
  3. 晚期希腊哲学

1.3 前苏格拉底哲学

  • 希腊人认为自然的构成有四大元素：水、火、土、气

    哪种元素是最古老的开端？产生了↓

  • 西方哲学的第一个概念：本原(arche)。本原的两大哲学含义：开端 主宰

    所谓本原(arche) 就是万物从它那里来，毁灭之后又回到它那里去，万物生灭变化，唯独它不变的东西。——亚里士多德

    本原实质上解决的是关于自然的知识问题，如何对自然形成具有普遍性和必然性的知识。

  • 泰勒斯的观点：

    1. 大地浮在水上→水是万物的本原。

      这里其实是比喻，并非江河湖海里的水。以水的流动性和可塑性来比喻本原应该具有的特征。（古代没有那么多抽象哲学概念可供使用，只能以自然存在的事物来象征）

    2. 宇宙充满了灵魂。

  • 赫拉克利特的观点：

    这个万物自同的宇宙，既不是任何神，也不是任何人所创造的，它过去是、现在是、将来也是一团永恒的活生生的火，按照一定的分寸燃烧，按照—定分寸熄灭。——赫拉克利特

    →火是万物的本原。

    赫拉克利特更加强调变化，自然界中只有变是不变的，但是变化是有规律的。

    人不能两次踏入同一条河流。——赫拉克利特

    一切皆流 无物常驻——柏拉图

  • 早期自然哲学的局限：

    1. 缺乏自然科学的支持。
    2. 建立在经验观察的基础上。
    3. 众说纷纭，莫衷一是。
    4. 无穷后退不可能。

1.4 本体论的转向

  • 毕达哥拉斯学派：更倾向于有确定性的东西作为知识的基础。

    宇宙万物的本原是数。——毕达哥拉斯

    相对于局限于经验观察的角度（水火土气谁是本原？），其更具有抽象性和普遍性。

  • 爱利亚学派-巴门尼德：赫拉克利特强调流变；巴门尼德强调我们要认识的对象是不动不变的。

    • 意见之路：非存在（认为非存在存在，存在反倒不存在）

      巴门尼德看来泰勒斯或赫拉克利特的观点只是意见而不能获得知识。

    • 真理之路：存在（认为存在存在，非存在不存在，以存在为对象才是真理之路）

    作为思想和作为存在是一回事→思维与存在的同一性，因为存在只有思想才能够把握，非存在（感觉经验的对象）既不能被诉说也不能被思想。存在是不动不变的，这才是哲学应该追问的对象。

  • 存在是西方哲学的核心概念。“存在”的概念提出有西方语言的必然性（系动词–是），奠定了西方哲学的核心部门形而上学的研究对象。

  • 巴门尼德的贡献：

    1. 确定了形而上学的研究对象——存在。
    2. 确定了思辨哲学的一个基本公式：思维与存在的同一性。
    3. 在哲学中开始了推理论证。

  • 不同思路

    • 泰勒斯和赫拉克利特：从构成事物的材料、从时间上追求关于自然的知识
    • 巴门尼德：从构成事物的本质规定、概念定义的角度追求关于自然的知识
